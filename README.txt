In order to compile this code it is necessary use Tegra Android Development Pack 
from NVIDIA. It can be downloaded through the link: 
https://developer.nvidia.com/gameworksdownload#?dn=tegra-android-development-pack-3-0r4

This app uses OpenCV features and needs the library cardboard.jar available for download in
the link: 
https://github.com/googlesamples/cardboard/blob/master/CardboardSample/libs/cardboard.jar
